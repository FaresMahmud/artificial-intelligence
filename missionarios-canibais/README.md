# Missionários e Canibais

Implementação em Java do problema clássico Missionários e Canibais usando busca em largura (BFS) para encontrar a solução e reconstruir o caminho.

## Estrutura

- `src/main/java/probMissionariosCanibais/Estado.java`
- `src/main/java/probMissionariosCanibais/Main.java`

## Como executar

1. Abra o projeto no IntelliJ IDEA.
2. Rode a classe `probMissionariosCanibais.Main`.
