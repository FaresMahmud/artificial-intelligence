# Artificial Intelligence

## Conteúdo

### aula.py
Arquivo com anotações sobre problemas e desafios em Algoritmos Genéticos (AG):

1. **Muita restrições** - Limitações impostas ao problema
2. **Não ter ideia do estado final** - Desconhecimento do objetivo final
3. **Não ter ideia de como chegar no estado final** - Falta de caminho claro
4. **Não ter ideia de como avaliar o estado final** - Função de avaliação indefinida
5. **Não ter ideia de como avaliar o estado intermediário** - Métricas intermediárias desconhecidas
6. **Não ter ideia de como gerar os filhos do estado atual** - Operadores genéticos não definidos

## Descrição

Este repositório contém estudos e implementações sobre Inteligência Artificial, com foco em Algoritmos Genéticos e suas aplicações.

Repositório para projetos de IA e problemas clássicos.
